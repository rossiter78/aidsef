# Retrospectives

One file per completed **feature** (not per task): `<feature>.md`. Written by the **Retro agent** in Phase 6 (`/aidsef-6-retro`), mining the feature's history — recurring review findings, human triage overrides, escalation rate, red-proof failures, coverage and mutation trends.

When the evidence shows an actionable pattern (at least two occurrences), the Retro agent opens a **separate** pull request proposing an amendment to [`constitution.md`](../constitution.md) or a role charter. Amendment PRs are always human-approved (constitution §8) — the framework improves itself under the same governance it applies to code.
